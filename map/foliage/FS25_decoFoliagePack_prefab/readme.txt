[DE]Kurzes Wort vorab:

Das Paket ist ein Prefab welches selbst in die Karte eingebaut werden muss!

Wichtig: Jede Karte verwendet andere Ordnerstrukturen, dies muss beim Einbau zwingend beachtet werden! 
Es können auch nur die Texturen des jeweiligen Foliage genutzt werden dennoch müssen dann bei decoFoliageUS.i3d, grass.i3d & meadowUS.i3d XML pfade angepasst werden!

1. FS25_decoFoliagePack_prefab.zip entpacken
2. FS25_deineKarte.zip entpacken
3. FS25_decoFoliagePack_prefab/foliage Navigieren
4. FS25_deineKarte zu foliage Ordner Navigieren
5. Aus FS25_decoFoliagePack_prefab/foliage die Ordner decoFoliageUS, grass, meadowUS kopieren
6. In FS25_deineKarte bei foliage Ordner decoFoliageUS, grass, meadowUS aus FS25_decoFoliagePack_prefab einfügen
7. In FS25_deineKarte die map.i3d mit Notepad++ öffnen
8. In Notepad++ STRG + F drücken
9. Bei Feld Suchen nach: grass.xml, meadow(US,AS,EU), decoFoliage(US,AS,EU)
10. Entsprechende XML Pfade anpassen
11. In Notepad++ STRG + S drücken zum speichern
12. Notepad++ schließen und die Karte in 7zip packen

[EN] A quick note in advance:

Important: Each map uses different folder structures, so this must be taken into account during installation!
You can also use only the textures of the respective foliage, but the XML paths for decoFoliageUS.i3d, grass.i3d, and meadowUS.i3d will need to be adjusted.

1. Unzip FS25_decoFoliagePack_prefab.zip
2. Unzip FS25_yourMap.zip
3. Navigate to FS25_decoFoliagePack_prefab/foliage
4. Navigate to the foliage folder in FS25_yourMap
5. Copy the folders decoFoliageUS, grass, and meadowUS from FS25_decoFoliagePack_prefab/foliage
6. Paste the folder decoFoliageUS, grass, and meadowUS from FS25_decoFoliagePack_prefab into foliage in FS25_yourMap
7. Open map.i3d in FS25_yourMap with Notepad++
8. In Notepad++, press CTRL + F
9. In the Search for field, enter: grass.xml, meadow(US,AS,EU), decoFoliage(US,AS,EU)
10. Adjust the corresponding XML paths
11. Notepad++: Press CTRL + S to save.
12. Close Notepad++ and unzip the map using 7zip.
